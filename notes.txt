For parser: 
go install golang.org/x/tools/cmd/goimports@latest

To generate parser.y.go from parser.y (usig goyacc):
goyacc -p phi -o parser/parser.y.go parser/parser.y

See test coverage

go test -gcflags 'all=-N -l' -p 1 -cover -coverpkg $(go list ./... | sed 's/\n/,/g') -coverprofile 1.out  ./...
go tool cover -html 1.out
